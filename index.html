Welcome to LPA PDB diagnostics.

This code allows analysis of LPA simulations using PIC code. The supported input format is pdb (using cPickle in Python). This tool allows both fields and particles analysis. It ranges from simple instant per instant analysis to analysis of multiple instances. The greatest feature is the post-processing particle analysis.

Installation requirements.

The following is a list of python modules that is necessary for the installation of LPA PDB diagnostics.

  • cPickle
  • h5py
  • numpy
  • scipy
  • Ipython notebook (optional)
  • pandas (optional)

The installation is simple: Type python setup.py install in your command line

Quickstarts

Two ipython notebooks are included:

  • quickstart_one.ipynb: for an instant per instant analysis
  • quickstart_multiple.ipynb: for multiple instances anlysis
